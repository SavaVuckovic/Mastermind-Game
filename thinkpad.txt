## Strategy
1. Pick a single color, try it as a Guess
2. Look at colors state result
3. If colors == 0, change color pick and repeat from 1
4. Else put color as guess[0], and generate guess[1..5] another color
5. If position == 2
  - keep first color at position[0]
  - keep second color at position[1]
6. If position == 1
  - swap guess[0] with guess[1]
